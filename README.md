# Color-Identification-OpenCV
This project is a simple and effective computer vision application that identifies colors from an image based on mouse clicks.
## How It Works
When you double-click anywhere on the displayed image, the system extracts the RGB values of that specific pixel, calculates the closest match from a dataset of over 800 colors, and displays the color name along with its RGB values directly on the screen.

## 🛠️ Technologies Used
* **Python**
* **OpenCV:** For image rendering, GUI display, and mouse event handling.
* **Pandas:** For dataset manipulation and distance calculation.
* **NumPy:** For array operations.
